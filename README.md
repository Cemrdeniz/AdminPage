# Admin Dashboard – Frontend

## Overview
A responsive **React-based admin dashboard** for managing a medicine store system.  
The application allows administrators to monitor statistics and manage **products, orders, customers, and suppliers** through a structured dashboard interface.

---

## Tech Stack
- React
- React Router
- React Hook Form
- Yup (form validation)
- Material UI / Blueprint UI
- CSS Modules

---

## Features
- Authentication with validated login form
- Protected routes for authorized users
- Dashboard with system statistics and recent customer data
- Orders management with username filtering
- Products management with create, update, and delete functionality
- Suppliers management with modal-based editing
- Customers list with filtering and pagination
- Responsive layout with optimized assets and icon sprites

---

## Structure
- **SharedLayout** – Global layout containing the Header and Sidebar navigation
- **Dashboard** – Displays statistics, recent customers, and income/expenses
- **Orders** – Orders table with filtering functionality
- **Products** – Product list with add, edit, and delete functionality
- **Suppliers** – Supplier management with modal forms
- **Customers** – Customer data table with filtering and pagination
